📋 How to share via SharePoint / Teams

Step 1 — Upload this HTML file to SharePoint
Go to your SharePoint site → Documents → upload this file into a folder (e.g. M&A Tools).

Step 2 — Embed it on a SharePoint page
Click + New → Page in SharePoint. Add an Embed web part and paste the direct URL of this HTML file. Save and publish the page.

Step 3 — Pin it as a Teams tab
In your Teams channel, click + → choose SharePoint → select the page you just created. Now the tracker lives as a tab.

Step 4 — Keeping data in sync between editors
After making edits, click ⬇ Export CSV and share that file in the Teams channel. The next editor opens the tool, unlocks edit mode, clicks ⬆ Import CSV, and selects the latest CSV.
